      • Hover Off

      • Hover Off

      • Hover Off

      • Hover Off

  • Rules
    • Highlighted Text
      • Default State
        • Hover On
          • Cursor changes to select state
          • Highlight on text changes color to purple
        • Hover Off
          • Cursor changes to default state
          • Highlight on text changes color to gray
        • Click
          • Highlight on text changes color to purple
      • Clicked State
        • Hover On
          • Cursor changes to select state
        • Hover Off
          • Cursor changes to default state
        • Click
          • Highlight on text changes color to gray
    • Share Button
      • Default State
        • Hover On
          • Cursor changes to select state
          • Button text becomes bold
        • Hover Off
          • Cursor changes to default state
          • Button text returns to default state
      • Clicked State
        • Hover On
          • Cursor changes to select state
          • Button text becomes bold
        • Hover Off
          • Cursor changes to default state
          • Button text returns to default state
    • Copy Link Button
      • Hover On
        • Cursor changes to select state
        • Button background color changes to black
        • Button text color changes to white
      • Hover Off
        • Cursor changes to default state
        • Button background color changes to white
        • Button text color changes to black
      • Click
        • Text changes from "Copy" to "Copied" for 1.5 seconds before changing back to "Hover On" or "Hover Off" state depending where cursor is
        • Button background color changes to white and text color changes to black for 1.5 seconds before changing back to "Hover On" or "Hover Off" state depending where cursor is
        • Green checkmark appears above the share link in module for 1.5 seconds before disappearing
        • Link copies to user's clipboard
    • Copy Link Button
      • Hover On
        • Cursor changes to select state
        • Button background color changes to black
        • Button text color changes to white
      • Hover Off
        • Cursor changes to default state
        • Button background color changes to white
        • Button text color changes to black
      • Click
        • Text changes from "Copy" to "Copied" for 2 second before changing back to "Hover On" state
        • Button background color changes to white and text color changes to black for 2 second before changing back to "Hover On" state
        • Green checkmark appears on right side of button for 2 second
        • Link copies to user's clipboard
    • Contributor Label
      • Hover On
        • Cursor changes to select state
        • Text becomes bold
        • Label apears above text, stating contributor's name
      • Hover Off
        • Cursor changes to default state
        • Text returns to default state
        • Label above text disappears
    • Like Button
      • Default State
        • Hover On
          • Cursor changes to select state
          • Thumbs up becomes black
        • Hover Off
          • Cursor changes to default state
          • Thumbs up returns to white
        • Click
          • Cursor changes to default state
          • Thumbs up becomes black
          • Number besides thumbs up goes up by one
      • Clicked State
        • Hover On
          • Cursor changes to select state
        • Hover Off
          • Cursor changes to default state
        • Click
          • Cursor changes to default state
          • Thumbs up becomes white
          • Thumbs down returns to default state
          • Number besides thumbs up goes down by one
